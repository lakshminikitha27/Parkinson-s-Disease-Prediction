Early Detection Method for Parkinsons disease Using Deep Ensemble Learning on MRI Dataset

Get Started


Abstract

Parkinson's Disease (PD) is a progressive neurological disorder characterized by motor dysfunction. Early detection of PD is crucial for improving patient outcomes. This study leverages deep learning techniques to develop a robust classification model for Parkinson's Disease detection using ResNet50. Our model is trained on MRI images and deployed in a Flask web application to provide real-time diagnosis. The model achieves high accuracy through transfer learning from ResNet50 pre-trained on ImageNet and fine-tuned for this medical imaging task. The web application is equipped with a user-friendly interface to enable clinicians and researchers to upload MRI scans and receive predictions for PD in real-time.
